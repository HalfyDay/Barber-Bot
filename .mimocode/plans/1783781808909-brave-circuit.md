# Plan: Add barber shop address block to client shop

## Context
User wants the barbershop address displayed in two places on the client-facing website:
1. In "Мои заказы" (My Orders) tab of the shop
2. In the cart sheet (Корзина)

## Data source
Address data is already available in `state.payload?.site?.home`:
- `mapCaption` — address string (e.g. "Гагарина 10, центральный вход, повернуть направо")
- `mapLink` — 2GIS link (e.g. "https://go.2gis.com/fPKd6")
- `logoText` — shop name (e.g. "BrotherShop")
- `phone` — phone number

## Changes

### 1. `client-app.js` — Add address block helper + render in both places

**a) Add a reusable HTML builder** (near `renderShopPage`, ~line 4083):
```js
const buildShopAddressBlock = () => {
  const siteHome = state.payload?.site?.home || {};
  const address = normalizeText(siteHome.mapCaption);
  const mapLink = normalizeText(siteHome.mapLink);
  const name = normalizeText(siteHome.logoText);
  if (!address && !name) return '';
  return `
    <div class="shop-address-block">
      <div class="shop-address-icon">📍</div>
      <div class="shop-address-info">
        ${name ? `<strong class="shop-address-name">${name}</strong>` : ''}
        ${address ? `<span class="shop-address-text">${address}</span>` : ''}
      </div>
      ${mapLink ? `<a class="shop-address-map-link" href="${mapLink}" target="_blank" rel="noopener noreferrer">На карте</a>` : ''}
    </div>
  `;
};
```

**b) Insert in "Мои заказы" tab** (line ~4264):
Before `<div class="shop-orders-list">${ordersHtml}</div>`, add the address block:
```js
<div class="shop-orders-list">${buildShopAddressBlock()}${ordersHtml}</div>
```

**c) Insert in cart sheet** (`openShopCartSheet`, line ~5704):
In `cartSheetHtml`, add the address block before the items:
```js
const cartSheetHtml = `
  ${buildShopAddressBlock()}
  <div class="shop-cart-items">...
```
