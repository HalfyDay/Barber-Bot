**Target:** Replace with a `CustomSelect` dropdown for operation mode + number input. The dropdown options:
- `increase` — Пополнение (positive delta)
- `decrease` — Списание (negative delta)
- `set` — Установить значение (calculates delta from current stock)

**Implementation:**
- Add `stockAdjustMode` state (default: `'increase'`)

  - `decrease` → `delta = -quantity`
